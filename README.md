# 🥗 NutriGo – Healthy Food Delivery Web App

NutriGo is a modern, responsive web application designed to promote healthy eating by delivering fresh, nutritious meals to users.
This project showcases a clean UI/UX design inspired by real-world food delivery platforms.

---

## 🚀 Live Preview

https://nutrigo-healthy-food-delivery.netlify.app/

---

## 📌 Project Overview

NutriGo focuses on providing a seamless user experience for exploring meals, choosing plans, and understanding how a healthy food delivery service works.

It is built as a **frontend-focused project** to demonstrate strong design skills, layout structuring, and responsiveness.

---

## ✨ Features

* 🏠 Clean and modern landing page
* 🥗 Healthy meal showcase section
* 📊 Pricing plans (Starter, Pro, Premium)
* ⚙️ Process section explaining service flow
* 🌍 Multi-city availability section
* 💬 Customer testimonials
* 📩 Contact form
* 🎨 Fully responsive design

## ⚙️ Installation & Usage

1. Clone the repository:

```bash
git clone https://github.com/nk-aditya/NutriGo.git
```

2. Open the project folder:

```bash
cd NutriGo
```

3. Run the project:

* Open `index.html` in your browser
